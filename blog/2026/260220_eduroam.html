Configure Eduroam on Linux

[cli] [eduroam] [linux] [network]

To setup eduroam network on linux (after executing the script instalador de eduroam):

  • Use the GUI application “Configuración de red avanzada”
  • Select “eduroam” network
  • In tab “Seguridad inalámbrica” set “guardar para todos los usuarios”, fill the password and click on “guardar”
  • Finally, connect to the network using “nmtui” or something similar

The connection can be done through the cli (maybe it is a good idea to use a script)

nmcli d wifi connect eduroam
