Web Interaction Hub

[available] [teaching] [tfg]

The main goal of this TFG is the development of a Web application that allows the management of the interaction according to the proposal for distributing Web interaction capabilities by Albertos et. al. [1].

The Web Interaction Hub is a new approach for designing the distribution, among interactive capabilities, of the interactive elements of Web applications. This project has three main objectives:

  • Firstly, the development of the Web Interaction Hub Plugin for the management of Web interaction
  • Secondly, the development of a front-end for the design of the interaction on interactive systems (This part of the project has been assigned to a student)
  • Finally, the system will include a number of gamification aspects to foster the understanding of human-computer interaction in educational environments

This project requires high skills in the development of Web applications

[1]
Albertos-Marco, F. et al. 2016. Distributing web interaction capabilities: The web interaction hub. Proceedings of the XVII international conference on human computer interaction (New York, NY, USA, 2016), 37:1–37:8.
