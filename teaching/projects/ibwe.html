Improved BFP Web Editor

[available] [teaching] [tfg]

The main goal of this TFG is the improvement of a Web application that supports the Template for the Bachelor’s Final Project (TFG), allowing the edition and generation of TFG documents.

The previous project, TFG Web Editor, is a Web platform for the management, edition and production of TFG documents within a friendly Web environment.

This projects it’s aimed to improve how this Web Application works, allowing local management of assets and asynchronous synchronization of the remote data base. To achieve this, Service Workers are expected to be used, so be ready to learn a lot of one, among others, of the most interesting technologies in nowadays Web Applications technological stack.

This project requires high skills in Web Applications
