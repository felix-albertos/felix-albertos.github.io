TFG Web Editor

[available] [teaching] [tfg]

The main goal of this TFG is the development of a Web application that supports the Template for the Bachelor’s Final Project (TFG), allowing the edition and generation of TFG documents.

TWE, Template Web Editor, is a Web platform for the management, edition and production of TFG documents within a friendly Web environment. Currently, these tasks are performed manually, editing text documents and executing scripts over the command line interface. With this platform, all this process will be carried out through the Web, in an usable manner.

This project requires high skills in both Web and Command Line Interfaces
